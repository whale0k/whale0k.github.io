Java基础语法

第一个程序

  • 编译：javac HelloWorld.java（将 java 源文件编译为 class 字节码文件）

  • 执行：java HelloWold

执行了第一个java程序后，了解下具体含义

Java关键字

  1. abstract：用于声明抽象类或抽象方法。
  2. assert：用于进行断言检查，用于调试和测试。
  3. boolean：表示布尔类型。
  4. break：用于跳出循环或switch语句。
  5. byte：表示字节类型。
  6. case：用于定义switch语句中的分支。
  7. catch：用于捕获异常。
  8. char：表示字符类型。
  9. class：用于定义类。
  10. const：已废弃，不再使用。
  11. continue：用于跳过循环的当前迭代。
  12. default：用于定义switch语句中的默认分支。
  13. do：用于定义循环的起始部分。
  14. double：表示双精度浮点类型。
  15. else：用于定义if语句中的分支。
  16. enum：用于定义枚举类型。
  17. extends：用于声明类的继承。
  18. final：用于修饰类、方法或变量，表示不可修改。
  19. finally：用于定义finally块，用于无论是否抛出异常都会执行的代码。
  20. float：表示单精度浮点类型。
  21. for：用于定义循环。
  22. if：用于定义条件语句。
  23. implements：用于实现接口。
  24. import：用于导入其他包中的类。
  25. instanceof：用于判断对象是否属于某个类的实例。
  26. int：表示整数类型。
  27. interface：用于定义接口。
  28. long：表示长整数类型。
  29. native：用于声明一个本地方法，即由非Java代码实现的方法。
  30. new：用于创建对象。
  31. package：用于定义包。
  32. private：表示私有访问修饰符。
  33. protected：表示受保护访问修饰符。
  34. public：表示公共访问修饰符。
  35. return：用于从方法中返回值。
  36. short：表示短整数类型。
  37. static：表示静态访问修饰符。
  38. strictfp：用于定义浮点运算的精确规则。
  39. super：用于引用父类的成员。
  40. switch：用于定义多分支选择语句。
  41. synchronized：用于实现线程同步。
  42. this：用于引用当前对象。
  43. throw：用于抛出异常。
  44. throws：用于声明方法可能抛出的异常。
  45. transient：用于声明不需要序列化的字段。
  46. try：用于定义异常处理块。
  47. void：表示无返回值类型。
  48. volatile：用于声明易变字段，确保多线程正确处理。
  49. while：用于定义循环。

常量

定义变量的时候，如果加上final修饰符，这个变量就变成了常量

常量在定义时进行初始化后就不可再次赋值，再次赋值会导致编译错误

常量的作用是用有意义的变量名来避免魔术数字（Magic number），例如，不要在代码中到处写3.14，而是定义一个常量。如果将来需要提高计算精度，我们只需要在常量的定义处修改，例如，改成3.1416，而不必在所有地方替换3.14。

根据习惯，常量名通常全部大写。

Java数据类型

基本数据类型

  1. 整数类型：包括byte（字节，8位）、short（短整型，16位）、int（整型，32位）和long（长整型，64位）。

  2. 浮点类型：包括float（单精度浮点型，32位）和double（双精度浮点型，64位）。

  3. 字符类型：char（字符类型，16位），用于表示单个Unicode字符。

  4. 布尔类型：boolean（布尔类型），用于表示真或假。

Java的运算符

  1. 算术运算符：用于基本的数学运算，如加（+）、减（-）、乘（*）、除（/）和模（%）。
  2. 关系运算符：用于比较两个值之间的关系，返回布尔值（true或false）。包括等于（==）、不等于（!=）、大于（>）、小于（<）、大于或等于（>=）和小于或等于（<=）。
  3. 逻辑运算符：用于组合布尔值，包括逻辑与（&&）、逻辑或（||）和逻辑非（!）。
  4. 位运算符：用于对二进制位进行操作，包括位与(&)、位或(|)、位非(~)、位异或(^)、左移(<<)、右移(>>)和无符号右移(>>>)。
  5. 赋值运算符：用于将值分配给变量，包括赋值（=）、加等号（+=）、减等号（-=）、乘等号（*=）、除等号（/=）、模等号（%=）、左移等号（<<=）、右移等号（>>=）和无符号右移等号（>>>=）。
  6. 条件运算符（三目运算符）：格式为 条件 ? 值1 : 值2，如果条件为真则返回值1，否则返回值2。
  7. 类型转换运算符：包括强制类型转换（cast）和自动类型转换。
  8. 空运算符：用于判断对象是否为空，返回布尔值。
  9. 正则表达式运算符：用于处理正则表达式，如匹配（=）、不匹配（!）和查找下一个匹配项（g）。
